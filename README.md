# MP4 to MP3 Converter

Convert `.mp4` video files to `.mp3` audio using Python and [moviepy](https://github.com/Zulko/moviepy).

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python mp4_to_mp3.py input_video.mp4 [output_audio.mp3]
```

If `output_audio.mp3` is not provided, the script will save the audio with the same base name as the input video.
